定时任务

设置 Agent 在指定的时间自动干活，比如每天早晨发报告、定期检查消息。每个任务可以单独开关，也可以随时修改。

技能（Skills）

在这里管理 Agent 能使用的技能文件。技能是 Markdown 文档，描述了 Agent 可以做什么、怎么做。勾选即可启用/停用，修改后保存，下次新会话生效。

工具

集成外部工具，开启后 Agent 可以使用对应能力。每个工具可以独立开关，也可以编辑配置。

系统提示（Agent 配置文档）

点击文件名即可编辑对应文档的内容。所有文档默认参与系统提示，无需额外勾选。保存后下一轮对话生效。
