多模型预设

按能力添加模型：先选用途（对话 / 识图 / 生图…），再选服务商并填写连接信息。自部署（Ollama、OpenAI 兼容等）与云厂商使用同一表单，保存后自动标记能力，无需手改 JSON。聊天页「模型栈」中选用各能力 preset。

MCP 服务

管理 Model Context Protocol 服务。支持 stdio（本地子进程）和 SSE（远程 HTTP）两种传输协议。 Agent 通过 mcp__服务ID__工具名 调用（需开启「注册给 Agent」）。
