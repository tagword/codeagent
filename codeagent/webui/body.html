聊天与会话

在这里调整聊天的行为，比如对话太长时自动总结、工具调用超出上限后自动继续等。改完点"保存"就能生效。

0 表示不限，直到任务完成或被手动停止

多模型预设

设置不同的 AI 模型，聊天时可以在输入框上方快速切换。每个预设可以配置接口地址、模型名和密钥。点击卡片即可展开编辑。

技能（Skills）

            

              
              
            

          

          

            在这里管理 Agent 能使用的技能文件。技能是 Markdown 文档，描述了 Agent 可以做什么、怎么做。勾选即可启用/停用，修改后保存，下次新会话生效。
